# 太极OS Dashboard - UI 设计指南

**版本：** v1.0  
**最后更新：** 2026-03-10

---

## 设计理念

太极OS Dashboard 的设计遵循以下核心理念：

1. **清晰直观** - 信息层次分明，用户能快速找到所需内容
2. **专业可靠** - 体现系统的稳定性和专业性
3. **高效操作** - 减少点击次数，提高操作效率
4. **响应式设计** - 适配桌面、平板、移动端

## 色彩方案

### 主色调
```
主色（Primary）: #1890ff (蓝色)
  - 用于：主要按钮、链接、重点信息
  - RGB: 24, 144, 255
  - 使用场景：导航栏、主要操作按钮、图表主色

辅助色（Secondary）: #722ed1 (紫色)
  - 用于：次要按钮、标签、辅助信息
  - RGB: 114, 46, 209
  - 使用场景：次要操作、特殊标记
```

### 状态色
```
成功（Success）: #52c41a (绿色)
  - 用于：成功状态、完成标记
  - RGB: 82, 196, 26
  - 使用场景：completed 状态、成功率高、健康状态

警告（Warning）: #faad14 (橙色)
  - 用于：警告状态、需要注意的信息
  - RGB: 250, 173, 20
  - 使用场景：warning 状态、中等优先级

错误（Error）: #f5222d (红色)
  - 用于：错误状态、失败标记
  - RGB: 245, 34, 45
  - 使用场景：failed 状态、critical 告警

信息（Info）: #1890ff (蓝色)
  - 用于：信息提示、普通状态
  - RGB: 24, 144, 255
  - 使用场景：info 级别、pending 状态
```

### 中性色
```
文字主色: #262626 (深灰)
文字次色: #595959 (中灰)
文字辅助: #8c8c8c (浅灰)
边框色: #d9d9d9 (边框灰)
```

## 动画规范

### 过渡时间
```
快速: 0.1s
正常: 0.3s
慢速: 0.5s
```

### 缓动函数
```
标准: cubic-bezier(0.4, 0, 0.2, 1)
进入: cubic-bezier(0, 0, 0.2, 1)
退出: cubic-bezier(0.4, 0, 1, 1)
```

### 常用动画
```css
/* 淡入淡出 */
.fade-enter {
  opacity: 0;
}
.fade-enter-active {
  opacity: 1;
  transition: opacity 0.3s;
}

/* 滑入滑出 */
.slide-enter {
  transform: translateY(-10px);
  opacity: 0;
}
.slide-enter-active {
  transform: translateY(0);
  opacity: 1;
  transition: all 0.3s;
}
```

## 可访问性（Accessibility）

### 颜色对比度
- 正文文字与背景对比度 >= 4.5:1
- 大文字（18px+）与背景对比度 >= 3:1

### 键盘导航
- 所有交互元素支持 Tab 键导航
- 支持 Enter/Space 键触发操作
- 支持 Esc 键关闭弹窗

### 屏幕阅读器
- 所有图标提供 aria-label
- 表单元素提供 label
- 动态内容提供 aria-live
