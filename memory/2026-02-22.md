## 模型成本优化 + 智能路由 (2026-02-22)
- 默认模型从 opus-4-6 切换到 sonnet-4-5（成本降低约5倍）
- 后备模型：opus-4-6（sonnet 失败时自动切换）
- 智能路由系统：
  - model_router.py：根据任务复杂度自动判断 sonnet/opus
  - task_dispatcher.py：简单任务当前会话处理，复杂任务自动 spawn opus 子 agent
  - 触发词库：重构/架构/调试/创建项目等 → opus，列出/查看/检查 → sonnet
  - 判断维度：关键词匹配、消息长度、代码块、上下文失败率
- 预期效果：日常对话省钱，复杂任务保质
- 路径：aios/core/model_router.py + task_dispatcher.py
