### 教训
- pyproject.toml 的 license 字段格式变了（不能用 `{text = "MIT"}`，直接用 `"MIT"`）
- License classifier 也要去掉（新版 setuptools 警告）
- Windows PowerShell 不支持 `&&`，要用 `;`
- Git 路径要用完整路径（`C:\Program Files\Git\bin\git.exe`）
