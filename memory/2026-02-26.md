## 第一次 GitHub 学习（02:44）

### 发现的项目

**AIOS: AI Agent Operating System** - 5.2K ⭐
- 仓库：https://github.com/agiresearch/AIOS
- 论文：https://arxiv.org/abs/2403.16971
- 文档：https://docs.aios.foundation/
- 更新：2025-01-22
- 语言：Python

**核心思路：**
1. **LLM 嵌入操作系统** - 把 LLM 当作 OS 的一部分
2. **AIOS Kernel + AIOS SDK** - 内核负责资源管理，SDK 负责 Agent 开发
3. **资源管理** - Scheduling、Context Switch、Memory、Storage、Tool、Agent SDK
4. **多种部署模式** - Local Kernel、Remote Kernel、Personal Remote Kernel、Virtual Kernel
5. **支持多种 Agent 框架** - ReAct、Reflexion、OpenAGI、AutoGen、MetaGPT
6. **支持多种 LLM** - OpenAI、Anthropic、Gemini、Groq、HuggingFace、Ollama、vLLM

**架构亮点：**
- **LLM Core(s)** - 多个 LLM 核心并行
- **Context Manager** - 上下文管理
- **Memory Manager** - 内存管理
- **Storage Manager** - 存储管理
- **Tool Manager** - 工具管理（包括 VM Controller 和 MCP Server）
- **Scheduler** - 调度器（FIFO、Round Robin）

### 对比我们的 AIOS

**我们的优势：**
1. ✅ 零依赖，可打包可复制（他们需要很多依赖）
2. ✅ 自我进化闭环（Self-Improving Loop）- 他们没有
3. ✅ 事件驱动架构（EventBus + Scheduler + Reactor）- 更灵活
4. ✅ 实时监控和自动修复（Reactor）- 他们没有

**我们的缺口：**
1. ❌ 社区生态（5.2K stars vs 我们 0 stars）
2. ❌ 文档完整性（他们有完整文档站）
3. ❌ 多种部署模式（他们支持 4 种模式）
4. ❌ Agent 框架集成（他们支持 5+ 框架）
5. ❌ LLM 支持（他们支持 9+ 提供商）
6. ❌ 学术认可（COLM 2025、NAACL 2025、ICLR 2025）

**可以借鉴：**
1. **AIOS Kernel + SDK 分离** - 内核和 SDK 分开，降低耦合
2. **多种部署模式** - Local、Remote、Personal、Virtual
3. **Agent 框架集成** - 支持 ReAct、Reflexion、AutoGen 等
4. **LLM 抽象层** - 统一接口支持多种 LLM
5. **Terminal UI** - LLM-based semantic file system
6. **Computer-use Agent** - VM Controller + MCP Server
7. **学术路线** - 发论文、进顶会、建立学术影响力
