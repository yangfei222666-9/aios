# autoresearch 项目工程尽调清单

- [ ] 是否有 `scheduler/` `state/` `retry/` `memory/` `audit/` 相关模块

### 3. 任务单位
- [ ] 任务单位是什么：hypothesis / experiment / run / trial？

### 4. 状态机
- [ ] 是否有显式状态机

### 5. 实验循环
- [ ] 循环机制：scheduler / search loop / evaluator？
- [ ] 如何生成新实验：手动 / 自动生成 / 搜索算法？
- [ ] 如何决定下一个实验：随机 / 优先级 / 启发式？
- [ ] 循环终止条件：固定次数 / 收敛 / 超时？

### 6. 结果存储
- [ ] 结果存储方式：db / jsonl / csv / files？
- [ ] 结果结构：包含哪些字段（输入、输出、指标、时间戳）？
- [ ] 是否支持结果对比和可视化

### 7. 失败处理
- [ ] 是否有 retry 机制
- [ ] retry 策略：固定次数 / 指数退避 / 条件重试？
- [ ] timeout 设置
- [ ] 失败后是否有 recover / rollback
- [ ] 失败是否记录到日志 / 数据库

### 8. "自动"的边界
- [ ] 自动生成实验假设？
- [ ] 自动编写实验代码？
- [ ] 自动运行代码？
- [ ] 自动评估结果？
- [ ] 自动调整参数？
- [ ] 还是只是自动调 prompt / API？
