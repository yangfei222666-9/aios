### 核心发现

- ⚠️ 缺少工具注册中心：工具定义分散在各个 Agent 中

2. **tool_cli.py** — 工具管理 CLI

   - 查看统计信息

### 教训与规则
**关键配置变更规则（永久执行）：**
1. **一次只动一个** — 如果有多个 bot/助手，一个改配置时另一个待命审查
2. **改之前先备份** — 确保能快速回滚（OpenClaw 自动备份到 .bak）
3. **改完先验证再重启** — 用 `openclaw doctor` 检查配置有效性
4. **不能两个一起挂** — 至少保证一个能正常工作，避免完全失联
5. **配置项要放对位置** — 不熟悉的配置项先查文档，不要随意添加
