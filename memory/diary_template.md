# 日记模板 v1.0

## 📅 时间
- **日期：** YYYY-MM-DD
- **时段：** 早晨 | 中午 | 下午 | 晚上 | 深夜

## 👥 人物
- 我
- 你（小九）
- 他/她
- 亲人
- 他人

## 📖 事件
- **原因：** 为什么发生
- **过程：** 怎么发生的
- **结果：** 最终怎么样

## 💭 核心情绪（选1-2个）
- 😊 高兴
- 😢 难过
- 😔 委屈
- 😠 生气
- 💬 倾诉
- 😴 疲惫
- 💪 坚定
- 😐 常态

## 🎯 关键观点
1. **自我感受：** 我怎么想
2. **他人视角：** 别人怎么看
3. **态度/规划：** 我打算怎么做
4. **核心观点：** 最重要的一句话

## 🔖 意图/结论（选1个）
- 🤝 求共情/陪伴（情感需求）
- 💡 求建议/表决心（行动需求）
- 📝 记事项/纯聊天（功能需求）

---

## 使用说明

**快速记录：**
```
2026-03-06 晚上 | 我+小九 | 讨论模板优化 | 坚定 | 求建议
核心：结构化记录工具，集成到AIOS记忆系统
```

**详细记录：**
按模板逐项填写，适合重要事件或复杂情绪
