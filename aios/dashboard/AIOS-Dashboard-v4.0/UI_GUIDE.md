## 颜色方案

### 主色调
- **背景色**：深蓝灰（#0f172a）
- **卡片背景**：中蓝灰（#1e293b）
- **边框**：灰蓝（#334155）

### 品牌色
- **青色**：#22d3ee（主要强调色）
- **翠绿**：#34d399（次要强调色）
- **渐变**：青色 → 翠绿（135deg）

### 状态色
- **成功/活跃**：翠绿（#34d399）
- **警告/空闲**：黄色（#fbbf24）
- **错误**：红色（#ef4444）
- **信息**：青色（#22d3ee）

### 文字色
- **主文字**：浅灰（#e2e8f0）
- **次要文字**：中灰（#94a3b8）
- **弱文字**：深灰（#64748b）

## 交互效果

### 1. 侧边栏折叠
- 点击左上角 [☰] 按钮
- 侧边栏宽度：250px → 60px
- 只显示图标，隐藏文字
- 平滑动画（0.3s）

### 2. KPI 卡片悬停
- 向上浮动 5px
- 显示顶部青绿渐变条
- 投影增强（青色光晕）

### 3. Agent 表格行悬停
- 背景变为半透明青色（rgba(34, 211, 238, 0.05)）

### 4. 日志自动滚动
- 新日志追加到底部
- 自动滚动到最新
- 最多显示 50 条

### 5. 实时更新
- 每 3 秒自动刷新数据
- 图表平滑更新（无闪烁）
- KPI 数字动态变化

## 响应式断点

### 桌面（> 1200px）
- 侧边栏固定显示
- KPI 卡片 4 列
- Skills 卡片 3 列

### 平板（768px - 1200px）
- 侧边栏可折叠
- KPI 卡片 2 列
- Skills 卡片 2 列

### 手机（< 768px）
- 侧边栏默认隐藏（点击显示）
- KPI 卡片 1 列
- Skills 卡片 1 列
- 搜索框宽度缩小

## 图表说明

### 系统健康度图表（Chart.js）
- **类型**：双线图
- **数据**：
  - 青色线：Evolution Score（90-100）
  - 翠绿线：成功率（90-100）
- **X 轴**：时间标签（12 个数据点）
- **Y 轴**：百分比（90-100）
- **更新**：每 3 秒刷新

## 性能优化

- 单文件架构（无需额外资源加载）
- CDN 加速（Bootstrap、Chart.js）
- 最小化 API 调用（3 秒间隔）
- 图表增量更新（'none' 模式）
- 日志限制 50 条（防止内存溢出）
