# ============================================
# 安装说明
# ============================================

# 基础安装（最小依赖）：
#   pip install -r requirements.txt
#
# 完整安装（包含可选依赖）：
#   pip install -r requirements.txt
#   pip install fastapi uvicorn prometheus-client
#
# Apple Silicon 优化：
#   # 使用 Apple 优化的 PyTorch
#   pip install torch torchvision torchaudio
#
# Intel Mac：
#   # 使用 CPU 版本
#   pip install torch --index-url https://download.pytorch.org/whl/cpu
#
# 虚拟环境（推荐）：
#   python3.12 -m venv venv
#   source venv/bin/activate
#   pip install -r requirements.txt

# ============================================
# 已知问题
# ============================================

# 1. torch 在 Apple Silicon 上可能需要特殊安装
#    解决：使用 Apple 优化版本或 CPU 版本
#
# 2. sentence-transformers 首次运行会下载模型
#    解决：预下载模型到 ~/.cache/huggingface/
#
# 3. lancedb 可能需要 Rust 编译器
#    解决：brew install rust
