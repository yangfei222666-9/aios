### 1. ✅ shadow / disabled 是否还会被算进 offline 或 unhealthy

### 2. ✅ active agent 比例的分母，是否仍错误包含 non-routable agent

**修正前：**
- 分母包含所有 agent，包括 shadow/disabled
- 导致 active 比例被稀释

**修正后：**
- 分母仍是 total（30），但 active 只统计 routable agent（11）
- shadow（14）和 disabled（3）单独统计

**验证结果：**
- Total: 30 = Active(11) + Shadow(14) + Disabled(3) + Others(2)
- Active 比例：11/30 = 36.7%（真实可路由能力）

✅ **通过** - active 比例反映真实可路由能力
