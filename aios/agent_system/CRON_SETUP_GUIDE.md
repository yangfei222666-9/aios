## 如何启用定时任务

### 方法1：手动添加到OpenClaw配置（推荐）

1. 打开OpenClaw配置文件
2. 找到 `cron` 部分
3. 复制 `daily_tasks_cron.json` 的内容
4. 粘贴到配置文件中
5. 重启OpenClaw

### 方法2：使用Cron工具添加

```bash
# 使用OpenClaw的cron工具
openclaw cron add --file daily_tasks_cron.json
```

### 方法3：逐个添加

```bash
# 添加单个任务
openclaw cron add --name "系统健康检查" --schedule "0 * * * *" --text "执行任务: 检查AIOS系统健康度"
```

## Cron表达式说明

| 表达式 | 含义 | 示例 |
|--------|------|------|
| `0 * * * *` | 每小时整点 | 10:00, 11:00, 12:00... |
| `0 9 * * *` | 每天09:00 | 每天早上9点 |
| `0 9 * * 1` | 每周一09:00 | 每周一早上9点 |
| `*/30 * * * *` | 每30分钟 | 10:00, 10:30, 11:00... |

## 注意事项

1. **避免任务冲突**
   - 多个任务不要在同一时间执行
   - 建议错开5-10分钟

2. **资源控制**
   - 每小时任务要轻量（<1分钟）
   - 每天任务可以稍重（<10分钟）
   - 每周任务可以更重（<30分钟）

3. **成本控制**
   - 监控API调用成本
   - 设置每日成本上限
   - 优先级低的任务可以禁用

4. **错误处理**
   - 任务失败会自动记录
   - 可以设置重试机制
   - 关键任务失败会发送告警
