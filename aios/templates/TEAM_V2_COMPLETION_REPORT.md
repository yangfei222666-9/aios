### 1. 新增 3 个关键 Agent

#### 🔬 研究员（Researcher）
**Agent ID：** `researcher`  
**工作时间：** 09:00-18:00  
**汇报对象：** CEO

**核心价值：** 这是珊瑚海的核心需求！每天自动去 GitHub 学习。

**每日任务（6个）：**
1. **09:00-10:00** - 搜索 GitHub 最新项目
   - 输入：关键词（AIOS、Agent System、Self-Improving）
   - 输出：项目清单（按 Star 排序）
   - 示例："发现 5 个新项目：AutoGPT (50k stars)、LangChain (40k stars)"

2. **10:00-11:00** - 分析高 Star 项目
   - 输入：项目清单、README、代码
   - 输出：项目分析报告
   - 示例："分析 AutoGPT：核心架构是 Planning + Memory + Tools"

3. **11:00-12:00** - 对比 AIOS
   - 输入：项目分析报告、AIOS 架构
   - 输出：对比分析报告
   - 示例："AIOS 优势：EventBus 架构更清晰；缺口：Planning 模块不如 AutoGPT 强大"

4. **14:00-15:00** - 阅读论文和技术博客
   - 输入：论文列表、技术博客
   - 输出：论文摘要和洞察
   - 示例："阅读论文：《ReAct: Synergizing Reasoning and Acting in Language Models》"

5. **15:00-16:00** - 实验和验证
   - 输入：新技术、新方法
   - 输出：实验结果
   - 示例："实验：在 AIOS 中实现 ReAct 模式，结果：任务成功率提升 15%"

6. **16:00-17:00** - 撰写研究报告
   - 输入：今日研究成果
   - 输出：每日研究报告
   - 示例："今日发现：3 个值得学习的项目，2 个改进建议，1 个实验结果"

**关键指标：**
- 每周分析项目数 > 10
- 每周阅读论文数 > 3
- 每月提出改进建议数 > 5
- 改进建议采纳率 > 50%
