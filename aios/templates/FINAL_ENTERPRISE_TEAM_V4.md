## 🆕 v4.0 新增的 3 个关键角色

### 1. 🧠 Prompt/Agent 研究员

**核心价值：** 让 AIOS 中的 Agent 更聪明、更可靠

**每日任务（5个）：**
1. **09:00-10:00** - 分析 Agent 性能数据
   - 示例："发现 Agent 在复杂任务上成功率只有 60%"

2. **10:00-12:00** - 设计和优化 Prompt 模板
   - 示例："设计 ReAct Prompt，成功率提升到 85%"

3. **14:00-15:00** - A/B 测试 Prompt
   - 示例："新 Prompt 成功率 85% vs 旧 Prompt 60%，提升 25%"

4. **15:00-16:00** - 研究前沿技术
   - 示例："研究 Reflexion：Self-Reflection 机制"

5. **16:00-17:00** - 构建 Prompt 模板库
   - 示例："添加 10 个新模板：代码生成、数据分析等"

**Prompt 模式：**
- Chain-of-Thought (CoT)
- ReAct (Reasoning + Acting)
- Reflexion (Self-Reflection)
- Tree-of-Thoughts
- Self-Consistency

**关键指标：**
- Agent 成功率提升 > 20%
- Prompt 模板数量 > 50
- A/B 测试次数 > 10/月

### 3. 🛡️ AI 安全与治理专家

**核心价值：** 让 AIOS 成为安全可信的 AI Agent 操作系统

**每日任务（5个）：**
1. **09:00-10:00** - 查看安全监控和告警
